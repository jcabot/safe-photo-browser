Google Drive Media Browser

Random Drive Media

Paste a Google Drive folder URL or folder ID to crawl that folder and every nested subfolder, then show one random image or video.

Media types

Access mode

Public folders work with an API key. For private or shared folders, configure OAuth and sign in first.

OAuth is optional unless the folder is not public.

Setup

  1. Create a Google Cloud project and enable the Google Drive API.
  2. Create a browser API key and set GOOGLE_API_KEY in .env.
  3. Optional: create a Web OAuth client and set GOOGLE_CLIENT_ID.
  4. If you use OAuth locally, add http://localhost:3000 to the authorized JavaScript origins.

Enter a Google Drive folder ID or URL, then load media.
